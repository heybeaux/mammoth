## Report decision

The first bounded change is to add a repeatable Metal throughput-and-latency mode
to the benchmark harness so the existing Metal backend and zero-copy unified
memory path can be measured before a kernel or memory change. The controlled
experiment uses paired repeated trials and rejects the change unless it preserves
outputs and clears the declared effect-size/confidence threshold.
